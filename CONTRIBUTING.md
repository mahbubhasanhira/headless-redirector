# Contributing to Headless Redirector

Thank you for your interest in contributing to Headless Redirector!

## Getting Started

1. Fork the repository.
2. Clone your fork locally.
3. Create a new branch for your feature or bug fix.

## Reporting Bugs

Please report bugs using the GitHub Issue Tracker. Include as much detail as possible.

## Pull Requests

1. Ensure your code follows WordPress Coding Standards.
2. Submit a Pull Request to the `main` branch.
